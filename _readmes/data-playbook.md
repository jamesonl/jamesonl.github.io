---
title: Data Activation Playbook
date: 2024-04-18
summary: Lightweight steps for turning raw telemetry into shared insight.
---

1. **Inventory sources.** Catalogue every dataset, owner, refresh cadence, and quality note.
2. **Define questions.** Use facilitation scripts to surface the top five decisions teams make weekly.
3. **Prototype loops.** Pair a single metric with narrative context and a retro prompt.
4. **Publish receipts.** Archive outputs in git so change history is visible and reversible.
