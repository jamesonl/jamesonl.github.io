Scenario Notes

The input above the globe changes the waterline directly. Once you move beyond a few dozen meters, the model is no longer acting like a conventional planning forecast and starts behaving more like a planetary stress test.

0 m · Present-day baseline

Modern projections are measured in fractions of a meter to a meter scale over this century, not in hundreds of meters. This baseline is here so the later jumps stay visually grounded.

    High slider values quickly move from climate projection into geological or speculative thought experiment territory.

    This application exaggerates land relief and simplifies coastlines into adaptive hex cells so changes remain readable. It is designed to provoke reasoning, not to serve as an engineering-grade inundation model.
